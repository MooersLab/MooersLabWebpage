    Protocol to create repository on github and git clone it

    • Create the repository on github: Autogenerate Readme and add license file.
    • git clone the repository locally.
    • Add gitignore file with gca command.
    • Add other files with the gca command.
    • git push origin master

    Protocol to edit a file locally and push the changes to GitHub

    • Edit the README.md file locally.
    • Add the changed file to repository with the command \mintinline{bash}{git add README.md}.
    • operation just stages the changed file.
    • Commit the changes with the command \mintinline{bash}{git commit -m "message about changes"}.
    • Send the changes to the GitHub site \mintinline{bash}{git push origin master}.
    • Must enter the github user name and password

His book has a website associated with it and listed below that provides access to over a 100 scripts. This is his second structural bioinformatics books. The first was more mathematical and more for computer scientists.
